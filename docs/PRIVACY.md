# Dates-LE Privacy Policy

## Overview

Dates-LE is committed to protecting user privacy and data security. This privacy policy outlines how we collect, use, and protect user information.

## Data Collection

### What We Collect

- **Local Telemetry**: Optional local-only telemetry data (disabled by default)
- **Usage Statistics**: Basic usage statistics for extension functionality
- **Error Reports**: Error information for debugging and improvement
- **Performance Metrics**: Performance data for optimization

### What We Don't Collect

- **Personal Information**: No personal information is collected
- **File Content**: No file content is transmitted or stored
- **User Data**: No user data is collected or transmitted
- **Sensitive Information**: No sensitive information is accessed
